# 去做自己的山 · Be Your Own Mountain

> 《去做自己的山》是一本关于自我成长与人生选择的书籍。
> 本项目以**逐页翻译 + 词汇讲解 + 朗读交互**的方式，帮助读者深入理解书中内容。

---

## ✨ 功能特性

- 📷 **OCR 文字识别**：支持从书籍照片中提取文字
- 🌍 **中英互译**：自动判断翻译方向（中文 → 英文 / 英文 → 中文）
- 📊 **三栏对照表**：
  - 原文（Source）
  - 译文（Translation）
  - 词汇与用法（Vocabulary & Usage）
- 🔊 **浏览器朗读交互**：
  - 点击译文 → 朗读整句
  - 点击单词/短语 → 仅朗读单词/短语本身
- 📝 **词性 / 音标 / 句中释义 / 常用短语** 一应俱全

---

## 📁 项目结构

```
.
├── README.md                                       # 本文件
├── .gitignore                                      # Git 忽略规则
├── Prompt.md                                       # 翻译与生成的总 Prompt
└── 去做自己的山.project/                           # 第 1 章子项目
    ├── Prompt.md                                   # 该章节使用的 Prompt
    ├── 去做自己的山-C01-...-P01.jpg                # 原始照片
    ├── 去做自己的山-C01-...-P02.jpg                # 原始照片
    ├── 去做自己的山-C01-...-S01-...-P01.html       # 生成的 HTML（带朗读交互）
    └── 去做自己的山-C01-...-S01-...-P01.md         # 生成的 Markdown 表格
```

---

## 🚀 使用方式

1. 用浏览器打开 `*.html` 文件
2. 第二列（蓝色）译文可点击，浏览器会用英语朗读整句
3. 第三列（橙色）单词/短语可点击，浏览器**只朗读该单词或短语本身**
4. 同名的 `*.md` 文件可作为纯文本对照表查看

---

## 🛠 技术栈

- 纯前端实现（HTML + CSS + 原生 JavaScript）
- 浏览器内置 [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/SpeechSynthesis) 实现朗读
- 无任何外部依赖，离线可用
